
📄 README – CPR (Controle de Pedidos e Romaneios)

🔹 Sobre o Programa  
O CPR foi desenvolvido para simplificar e automatizar o controle de pedidos, transporte e expedição de mercadorias, oferecendo funcionalidades como:

- Registro manual e automático de pedidos  
- Geração de romaneios de expedição e separação  
- Consulta de pedidos por nome ou nota  
- Registro de insumos via nota fiscal ou cupom  
- Emissão de relatórios de gastos mensais

🔹 Instruções Importantes  
✅ Atenção! Para o correto funcionamento do programa:

- NÃO renomeie ou mova os arquivos do programa de sua pasta original.  
- NÃO exclua arquivos como DB.txt, clientes.txt, transportadoras.txt, recebimentos.txt ou os arquivos de planilhas modelo.  
- O banco de dados é carregado diretamente a partir destes arquivos.  

📁 Os arquivos gerados automaticamente pelo programa (romaneios e relatórios) são salvos na pasta **"Romaneios de Expedição"** na sua Área de Trabalho.

🔹 Como Acessar o Programa  

1. Localize o arquivo **CPR.exe** na pasta onde o programa foi instalado.  
2. Crie um atalho para ele clicando com o botão direito > Enviar para > Área de trabalho (criar atalho).  
3. Use esse atalho para abrir o CPR com mais facilidade no dia a dia.  
4. Ao abrir, o terminal será exibido brevemente. Aguarde a janela principal branca do programa abrir.
